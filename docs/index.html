Fallen leaves

Fallen leaves is a simple slideshow viewer, the photos shows as falling leaves in the autumn.

Installation

Fallen leaves has several methods to install.

  • As a deb package for x86-64 Debian GNU/Linux and derivates (Ubuntu, Linux Mint...). You can install with gdeby or dpkg. Maybe in the future, it will be a PPA or official repository.
  • As a snap package for x86-64 GNU/Linux. You can download and install or install from snap store.

Usage

Config

Fallen leaves has a few setup options. They are the typical options of slideshow:

  • Delay between photos: in seconds (by default 4 seconds)
  • Sort images: Date, Name, Size (and reverse) (by default Name)
  • Or show the images random in random order

Command line

Who does not love the terminal? ...well the windows users and mac users... But I love the terminal and Fallen leaves has command line options: 


$ fallen-leaves -h
Fallen leaves is a simple slideshow viewer, the photos shows as falling leaves in the autumn.
/usr/lib/fallen-leaves/fallen-leaves <directory_images> <options>

Options:
   -d | --delay <number> : seconds delay between showing images
   -m | --max <number> : max photos in stage
   -s | --sort (alphabetic|date|size) : sort images
   -r | --reverse : flag to sort reverse images
   -z | --random : flag to sort random images

Usage

It extremely easy to use. Only you open image directory in topmenu entry File -> Open Dir (or shotcut Ctrl+O) and Fallen leaves starts showing the images.
